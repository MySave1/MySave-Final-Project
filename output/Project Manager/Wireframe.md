## 1. 초기 컨셉 및 와이어프레임 (Low-Fidelity)
기획 단계에서 구상한 서비스 흐름과 레이아웃 입니다. 크롬 확장 프로그램과 웹 대시보드 간의 유기적인 연결을 중점으로 설계했습니다.

[초기 와이어프레임 및 레퍼런스]<img width="1032" height="777" alt="와이어프레임 및 레퍼런스" src="https://github.com/user-attachments/assets/64ab1dc2-118e-45f0-95bf-eaa80b865402" />


### 📝 주요 설계 포인트
* **크롬 확장 프로그램:** 현재 보고 있는 웹페이지를 즉시 저장하고 태그/리마인드를 설정하는 팝업 UI
* **웹 대시보드:** 저장된 북마크를 카드 형태로 시각화하고 관리하는 메인 홈
* **리마인드 관리:** 사용자가 설정한 알림을 한눈에 보고 관리하는 기능
* **설정 및 태그:** 개인화된 설정을 위한 전용 페이지 구성

## 2. 최종 UI 디자인 (Final UI Design)
와이어프레임을 구체화하여 완성된 최종 UI 화면 구성도입니다. 사용자의 행동 흐름(User Flow)에 따라 화면을 설계했습니다.
