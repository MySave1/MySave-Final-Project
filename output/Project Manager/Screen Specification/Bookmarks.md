# 📑 북마크 목록 페이지 기획안

## 1. GNB (상단 메뉴)
| 항목 | 설명 |
| --- | --- |
| 메뉴 탭 | **[북마크 목록] 탭 활성화 (Active Style)** |
| 검색창 | 키워드 입력 시 **현재 북마크 리스트 내 검색 / 필터링** 동작 |

---

## 2. 필터 및 정렬 (Filter & Sort)
| 항목 | 설명 |
| --- | --- |
| 상태 필터 | `[전체] / [⭐ 즐겨찾기] / [읽음] / [안 읽음]` <br> - **Single Choice** <br> - 클릭 즉시 리스트 필터링 |
| 정렬 | `[최신순 (Default)] / [오래된순]` 드롭다운 제공 |

---

## 3. 목록 컨테이너
| 항목 | 설명 |
| --- | --- |
| 레이아웃 | **3열 그리드** (한 줄에 3개 카드 배치) |
| 반응형 | 화면 크기에 따라 **2열 / 1열** 자동 전환 |

---

## 4. 카드 아이템 (통합 기능)
### 4-1. 기본 정보
- 썸네일 (이미지 또는 컬러 박스)
- 타이틀 (최대 2줄, 말줄임 처리)
- 저장 날짜
- 태그 1개 노출

### 4-2. 상태 배지 (우측 상단)
| 상태 | 설명 |
| --- | --- |
| 요약됨 | 요약 완료 상태 표시 |
| 요약하기 | 클릭 시 **AI 요약 기능 실행** |

### 4-3. 관리 액션 (아이콘)
| 아이콘 | 기능 |
| --- | --- |
| ⭐ Star | 즐겨찾기 등록 / 해제 토글 |
| ✏️ Edit | 북마크 정보 수정 팝업 |
| 🗑 Delete | 삭제 확인 팝업 노출 |

### 4-4. 페이지 이동
- 카드 내 **아이콘 영역을 제외한 영역 클릭 시**
- **원본 URL 또는 북마크 상세 페이지로 이동**

---

## 5. 페이지네이션
| 항목 | 설명 |
| --- | --- |
| 구조 | `< 1 / 3 >` 형태 |
| 동작 | 페이지 번호 또는 화살표 클릭 시 리스트 갱신 |
