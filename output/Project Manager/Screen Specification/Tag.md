# MySave 태그(Tag) 화면 기획안

본 문서는 MySave 웹 서비스의 **태그 관리 화면 구성 및 기능 정의**를 목적으로 한다.  
사용자는 태그를 생성·관리함으로써 저장한 콘텐츠를 체계적으로 분류할 수 있다.

---

## 📌 태그 화면 구성 요소

| **NO** | **항목** | **Description** |
| --- | --- | --- |
| **1** | **GNB (상단 메뉴)** | a. **메뉴 탭**: `[태그]` 탭 활성화 (Active Style)<br><br> b. **검색창**<br> - 키워드 입력 시 **태그 목록 내에서 태그 이름 기준 검색** |
| **2** | **새 태그 추가 (Create Panel)** | **2-1. 태그명 입력**<br> - Placeholder: *"예: AI"*<br> - 유효성 검사: 공백 불가, **중복 태그명 생성 불가**<br> - 글자 수 제한: 최대 N자 (예: 10~15자 권장)<br><br> **2-2. 색상 태그 선택 (Color Picker)**<br> - 7가지 컬러 팔레트(원형) 제공<br> - 선택 방식: 라디오 버튼(Radio) 방식 (1개만 선택 가능)<br> - Default: 첫 번째 색상 또는 랜덤 선택<br><br> **2-3. [# 새 태그 만들기] 버튼**<br> - 클릭 시 유효성 검사 후 태그 생성<br> - 생성 성공 시 우측 **태그 관리 목록에 즉시 반영 (AJAX)** |
| **3** | **태그 관리 목록 (List Panel)** | a. **레이아웃**: 카드형 그리드 배치 (Flex / Grid)<br><br> b. **정렬 옵션**<br> - 글 개수 많은 순 / 가나다 순 (기획 의도에 따라 결정)<br><br> c. **스크롤 정책**<br> - 태그 개수 증가 시 해당 영역 내부 스크롤 또는 페이지 전체 스크롤 적용 |
| **4** | **태그 카드 아이템 (Unit)** | a. **디자인 스타일**<br> - 선택된 포인트 컬러와 매칭되는 **연한 배경색(Pastel Tone)** 자동 적용<br><br> b. **정보 노출**<br> - **태그명**: 굵은 폰트로 강조<br> - **포인트 점(Dot)**: 좌측에 원형 포인트 컬러 표시<br> - **글 개수**: 연결된 북마크 수 카운트 (예: *"5개의 글"*)<br><br> c. **클릭 이벤트 (Body)**<br> - 카드 클릭 시 해당 태그로 필터링된 **[북마크 목록] 페이지로 이동** |
| **5** | **태그 삭제 (Delete Action)** | a. **위치**: 태그 카드 우측 상단 `x` 버튼<br><br> b. **동작**<br> - 클릭 시 **삭제 확인 팝업(Alert)** 노출 (*"태그를 삭제하시겠습니까?"*)<br> - 확인 시 태그 삭제 및 목록 갱신<br><br> c. **데이터 처리 정책**<br> - 태그만 삭제되며 **북마크 데이터는 유지**<br> - 삭제된 태그는 해당 북마크와의 연결만 해제됨 |

---

## 🎯 태그 화면 설계 핵심 가치

- 사용자가 직접 만드는 **개인화 분류 체계**
- 색상 기반의 **시각적 구분 및 빠른 인식**
- 태그 중심 탐색을 통한 **콘텐츠 재발견 경험 강화**
