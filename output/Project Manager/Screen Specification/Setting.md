## 설정 페이지 기획안 v1

> **특징**
> - 기능 단위 중심의 단일 컬럼 구성
> - 설정 항목을 논리적으로 구분
> - 초기 MVP 및 1차 배포에 적합

### 📋 구성 요소

| **NO** | **항목** | **Description** |
| --- | --- | --- |
| **1** | **GNB (상단 메뉴)** | a. **메뉴 탭**: `[설정]` 탭 활성화 (Active Style)<br><br> b. **검색창**<br> - 설정 내 검색 또는 전체 글 검색 기능 (기획 의도에 따라 선택) |
| **2** | **프로필 설정 (User Profile)** | a. **프로필 이미지**<br> - 현재 이미지 노출 (없을 시 기본 아바타)<br> - 연필 아이콘 클릭 시 이미지 업로드 팝업 실행<br><br> b. **기본 정보**<br> - 닉네임, 이메일 주소 텍스트 노출<br><br> c. **계정 배지**<br> - 소셜 로그인 연동 상태 표시 (예: `Kakao 연동됨`)<br><br> d. **정보 수정 메뉴**<br> - 이메일 / 비밀번호 변경 링크 제공<br> - 소셜 로그인 유저는 비밀번호 변경 비활성화 등 예외 처리 |
| **3** | **AI 설정 (AI Configuration)** | a. **자동 요약 활성화**<br> - Toggle Switch (ON / OFF)<br> - ON (Default): 북마크 저장 시 3줄 요약 자동 생성<br> - OFF: 저장 시 요약 생략, 상세 페이지에서 수동 실행 |
| **4** | **환경 설정 (Preferences)** | a. **알림 설정**<br> - 리마인드 푸시 알림 Toggle<br> - 브라우저 알림 권한 허용 여부 제어<br><br> b. **디자인 설정 (Theme)**<br> - 다크 모드 Toggle<br> - OFF: 라이트 모드 (기본)<br> - ON: 다크 모드<br> - 설정값은 LocalStorage 또는 DB에 저장 |
| **5** | **데이터 관리 (Data Management)** | a. **북마크 가져오기**<br> - 브라우저 내보내기 파일(.html 등) 업로드 지원<br><br> b. **데이터 내보내기**<br> - `.csv` 또는 `.json` 파일 다운로드<br><br> c. **전체 데이터 삭제**<br> - Red Color 텍스트로 위험성 강조<br> - 경고 팝업 + 재확인(타이핑 입력 등) 필수 |
| **6** | **계정 연결 및 로그아웃 (Account Action)** | a. **소셜 연동 상태 버튼**<br> - `[Kakao 계정 연동됨]` 상태 표시<br> - 클릭 시 연동 해제 또는 재연동<br><br> b. **로그아웃 버튼**<br> - 세션 종료 후 로그인 페이지 이동 |

## 설정 페이지 기획안 v2

> **특징**
> - 카드(Card) 기반 레이아웃
> - 시각적 그룹화 강화
> - 데스크톱 사용성 및 확장성 향상

### 📋 구성 요소

| **NO** | **항목** | **Description** |
| --- | --- | --- |
| **1** | **GNB (상단 메뉴)** | a. **메뉴 탭**: `[설정]` 탭 활성화 (Active Style)<br><br> b. **검색창**: 상단 고정 노출 |
| **2** | **프로필 설정 (Top-Left Card)** | a. **프로필 이미지**: 원형 아바타 + 편집(연필) 버튼<br><br> b. **기본 정보**: 닉네임, 이메일 주소 노출<br><br> c. **상태 배지**: `Kakao 연동됨` 등 계정 상태 표시<br><br> d. **계정 관리 링크**<br> - `이메일 변경`, `비밀번호 변경` |
| **3** | **계정 연결 (Top-Right Card)** | a. **타이틀**: "계정 연결"<br><br> b. **정보 수정 버튼**<br> - `[닉네임/이메일 설정]` (Yellow Button)<br> - 클릭 시 정보 수정 모달 또는 페이지 이동<br><br> c. **로그아웃**<br> - 카드 하단에 아이콘 + 텍스트 링크 배치 |
| **4** | **AI 설정 (Bottom-Left Card)** | a. **타이틀**: "AI 설정" + 아이콘<br><br> b. **자동 요약 활성화**<br> - 설명: "북마크 저장 시 자동으로 내용을 3줄 요약합니다."<br> - Toggle Switch (ON / OFF) |
| **5** | **환경 설정 (Bottom-Center Card)** | a. **타이틀**: "환경 설정" + 톱니바퀴 아이콘<br><br> b. **알림 설정**<br> - 종(Bell) 아이콘 + 리마인드 푸시 알림 Toggle<br><br> c. **다크 모드**<br> - 해(Sun) 아이콘 + 다크 모드 Toggle |
| **6** | **데이터 관리 (Bottom-Right Card)** | a. **타이틀**: "데이터 관리" + 방패 아이콘<br><br> b. **기능 목록**<br> - `북마크 가져오기` (파일 업로드)<br> - `데이터 내보내기` (다운로드)<br> - `전체 데이터 삭제` (**Red Color** 강조) |
