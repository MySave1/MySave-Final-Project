## 배너 화면 기획안 v1
### (그리드형 카드 리스트)

| NO | 항목 | Description |
| --- | --- | --- |
| 1 | 목록 컨테이너<br>(List Area) | a. **레이아웃**: 2 x 2 그리드 형태 (한 페이지당 최대 4개 카드 노출)<br>b. **정렬**: 최신 저장순(내림차순) 기본 적용 |
| 2 | 카드 아이템<br>(UI 및 인터랙션) | a. **배경색**: 저장된 콘텐츠 속성에 따른 배경색 적용 (랜덤 또는 태그별 컬러)<br>b. **상태 배지**: 우측 상단 `요약됨` 칩(Chip) 노출 (요약 완료 시)<br>c. **타이틀**:<br>- 저장된 페이지 Title 노출<br>- 최대 2줄, 초과 시 말줄임(...)<br>d. **태그**:<br>- 입력된 태그 중 **첫 번째 태그 1개만 노출**<br>e. **클릭 이벤트**:<br>- 카드 클릭 시 **저장된 원본 URL 새 탭 실행** |
| 3 | 페이지네이션<br>(Pagination) | a. **구조**: `[이전] [현재 페이지 / 전체 페이지] [다음]`<br>b. **버튼 상태**:<br>- 이전: 첫 페이지에서 비활성화(Dimmed)<br>- 다음: 마지막 페이지에서 비활성화(Dimmed)<br>c. **동작**: 클릭 시 해당 페이지 데이터 갱신 |

## 배너 화면 기획안 v2
### (추천 위젯형 카드 리스트)

| NO | 항목 | Description |
| --- | --- | --- |
| 1 | 위젯 헤더 | a. **타이틀**: `추천 읽을거리` 고정 노출<br>b. **더보기 링크**: 우측 상단 `더보기 >`<br>- 클릭 시 전체 목록 페이지 이동 |
| 2 | 카드 리스트 영역 | a. **레이아웃**: 세로 나열형 (최대 2~3개 노출)<br>b. **스크롤**: 영역 초과 시 내부 스크롤 허용 가능<br>c. **노출 로직**:<br>- 랜덤 추천<br>- 또는 `과거의 오늘` 등 큐레이션 로직 |
| 3 | 카드 아이템<br>(Card UI) | a. **디자인 스타일**:<br>- 박스 형태 카드 UI<br>- 콘텐츠 태그/카테고리에 맞는 **파스텔톤 배경색**<br>- 라운드 처리(Border-radius)<br>b. **구성 요소**:<br>- **타이틀**: 상단 또는 중앙, 최대 2줄 말줄임<br>- **태그**: 하단 소형 텍스트 (공간 부족 시 생략 가능)<br>- **날짜**: 우측 하단 (예: 2024.12.20)<br>c. **상태**:<br>- 요약 배지 없음 (기능 미구현)<br>- 클릭 시 원본 URL 새 탭 이동 |
| 4 | 빈 상태<br>(Empty State) | a. **조건**: 저장된 북마크 0건<br>b. **UI**:<br>- 흰색 배경 박스<br>- 중앙 정렬 텍스트 **"저장된 글이 없습니다."** |

## 📌 버전 선택 가이드 (기획 참고)
- **v1**: 콘텐츠 노출량이 중요하고, 배너 자체가 메인 영역인 경우 적합
- **v2**: 사이드 위젯 또는 보조 콘텐츠 영역에 자연스럽게 녹이기 좋은 구조
