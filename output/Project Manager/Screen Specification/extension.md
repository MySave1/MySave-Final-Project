## 확장프로그램 화면 기획안 v1

| **NO** | **항목** | **Description** |
| --- | --- | --- |
| **1** | **헤더 (로고)** | a. **MY SAVE** 로고 노출<br> b. 클릭 시 메인 웹 대시보드(홈)로 새 탭 이동 (선택 사항) |
| **2** | **기본 정보 입력**<br>(URL, 제목, 텍스트) | a. **초기 진입 시 자동 입력 (Auto-fill)**<br> - 사용자가 현재 보고 있는 브라우저 탭의 URL, Page Title 자동 입력<br> - 사용자가 드래그(Highlight)한 텍스트가 있다면 ‘텍스트’ 영역에 자동 입력<br> b. **수정 가능**: 사용자가 내용을 직접 수정 가능 |
| **3** | **요약한 텍스트**<br>(AI 요약) | a. **Placeholder**: “AI가 내용을 요약해줍니다.” 문구 노출<br> b. **로직**<br> - (Case A) 팝업 로딩 시 자동 요약 실행<br> - (Case B) 별도 ‘요약하기’ 버튼 트리거 필요 (텍스트 입력 시 백그라운드 처리 또는 저장 시 처리) |
| **4** | **추가 정보**<br>(메모, 태그) | a. **메모**: 사용자 입력 텍스트 영역 (Multi-line)<br> b. **태그**<br> - 태그 입력 후 Enter 키 입력 시 태그 생성 (Chip 형태)<br> - 생성된 태그 옆 ‘x’ 버튼으로 삭제 가능 |
| **5** | **리마인드 설정**<br>(토글 스위치) | a. **Default**: OFF 상태 (좌측 회색)<br> b. **인터랙션**: 토글 클릭 시 ON / OFF 전환<br> c. **UI 변경**<br> - OFF: 하단 날짜 선택 옵션 영역 숨김<br> - ON: 하단 날짜 선택 옵션 영역 노출 |
| **5-1** | **리마인드 옵션**<br>(빠른 선택) | *리마인드 ON 상태에서만 활성화*<br> a. 빠른 설정 버튼 3종: `[내일 아침]`, `[이번 주말]`, `[다음 주]`<br> - 클릭 시 Active 스타일 적용<br> - 각 버튼 하단에 요일/시간 표시 (예: 09:00, 토요일) |
| **5-2** | **리마인드 옵션**<br>(직접 선택) | *리마인드 ON 상태에서만 활성화*<br> a. **직접 날짜/시간 선택하기** 버튼 제공<br> b. Date Picker & Time Picker로 사용자 지정 가능 |
| **6** | **저장하기 버튼** | a. **유효성 검사(Validation)**<br> - 필수 항목(URL 등) 미입력 시 저장 불가 및 알림 표시<br> b. **Action**<br> - URL, 제목, 텍스트, 요약, 메모, 태그, 리마인드 정보 서버 전송<br> - 저장 성공 시 “저장되었습니다” 토스트 팝업 노출 후 확장 프로그램 창 닫기 |
