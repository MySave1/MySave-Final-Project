## 🔹 북마크 상세 페이지 v1
> **기본 열람 + AI 요약 + 메모 제공 (초기 버전)**

| NO | 항목 | Description |
| --- | --- | --- |
| 1 | 네비게이션<br>(GNB & Back) | a. **GNB**: 상단 메뉴바 유지<br>b. **뒤로가기 (< 목록으로)**:<br>- 클릭 시 이전 목록 페이지 이동<br>- *목록의 필터/페이지 상태 유지 권장* |
| 2 | 메인 콘텐츠 영역<br>(Left Column) | a. **썸네일**: 원본 페이지 대표 이미지 (없을 시 대체 이미지)<br>b. **타이틀/링크**:<br>- 페이지 제목 노출<br>- 클릭 시 새 탭으로 원본 URL 이동<br>c. **메타 정보**:<br>- 즐겨찾기(⭐) 토글<br>- 태그 클릭 시 태그 검색 결과 이동<br>- 저장 날짜 표시<br>d. **본문 텍스트**: 스크롤 가능한 원본 텍스트 영역 |
| 3 | AI 요약 노트<br>(Right Widget 1) | a. AI 요약 텍스트 노출<br>b. **[다시 생성하기] 버튼**:<br>- AI 요약 API 재호출<br>- 로딩 중 Spinner 표시 |
| 4 | 메모<br>(Right Widget 2) | a. 메모 내용 표시 (없을 시 "메모가 없습니다")<br>b. **[메모 수정하기] 버튼**:<br>- Textarea 전환 또는 수정 모달<br>- 저장 시 내용 반영 |

## 🔹 북마크 상세 페이지 v2
> **UI 단순화 + 기능 단계적 비활성화 + 리마인드 추가**

| NO | 항목 | Description |
| --- | --- | --- |
| 1 | GNB 및 네비게이션 | a. 상단 대시보드 GNB 유지<br>b. `< 목록으로` Breadcrumb 제공 |
| 2 | 메인 콘텐츠 영역<br>(Left Column) | a. **타이틀 섹션**:<br>- 웹페이지 제목 (H1)<br>- 태그 Chip, 저장 날짜 표시<br>b. **본문 뷰어 (Card UI)**:<br>- 썸네일 제거<br>- 라운드된 흰색 카드 UI<br>- Empty State: "내용이 없습니다." |
| 3 | AI 요약 노트<br>(Right Widget 1) | **※ UI Only (미구현)**<br>- 타이틀: ✨ AI 요약 노트<br>- 기본 문구: "요약된 내용이 없습니다."<br>- `[요약 생성하기]` 버튼 (동작 없음) |
| 4 | 메모 위젯<br>(Right Widget 2) | a. **메모 없음**:<br>- Placeholder + `[메모 생성하기]` 버튼<br>b. **메모 있음**:<br>- 메모 텍스트 노출 |
| 5 | 리마인드 설정<br>(Right Widget 3) | a. 신규 기능 추가<br>b. **미설정**:<br>- 안내 문구 + `[리마인드 설정]` 버튼<br>c. **설정됨**:<br>- 날짜/시간 표시 + 수정 버튼 |

## 🔹 북마크 상세 페이지 v3
> **편집 중심 UX + 상태 관리 강화 (고급 버전)**

| NO | 항목 | Description |
| --- | --- | --- |
| 1 | 헤더 액션 그룹 | a. **편집 버튼**:<br>- `[수정하기]` ↔ `[저장 완료]`<br>- 편집 모드 전환<br>b. **읽음 상태 토글**:<br>- `[안 읽음]` ↔ `[읽음]`<br>c. **즐겨찾기**: Star 아이콘 유지 |
| 2 | 콘텐츠 에디터<br>(Left Column) | a. 텍스트 드래그 시 하이라이트 메뉴 (선택 사항)<br>b. **View / Edit 모드**:<br>- View: 읽기 전용<br>- Edit: 직접 수정 가능 |
| 3 | AI 요약 (미구현) | a. `[요약 생성하기]` 클릭 시:<br>- Disabled 상태 또는<br>- "서비스 준비 중입니다" 토스트 노출 |
| 4 | 메모 위젯 | a. `[메모 생성하기]` → Textarea 활성화<br>b. 저장 또는 포커스 아웃 시 뷰어 모드 전환 |
| 5 | 공통 UI 스타일 | - 전체 옅은 회색 배경<br>- 카드 UI: White + Radius + Shadow 적용 |
