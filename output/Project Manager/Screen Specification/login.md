# 🔐 계정 연결 로그인 페이지 (모달) 기획안

> 본 화면은 **설정 페이지에서 [계정 연결] 클릭 시 노출되는 로그인/계정 정보 설정 모달**입니다.  
> 소셜 로그인(Kakao) 연동 전 사용자 기본 정보를 입력받기 위한 UI를 제공합니다.

---

## 📋 UI 구성 요소 상세

| NO | 항목 | Description |
| --- | --- | --- |
| 1 | 모달 오버레이<br>(Background) | a. **스타일**: 블랙 계열 반투명 딤드(Dimmed) 처리<br>b. **동작**: 모달 외 영역 클릭 시 모달 닫힘<br>※ 정책에 따라 외부 클릭 차단 가능 |
| 2 | 모달 컨테이너 | a. **스타일**: 흰색 배경, 라운드 처리된 중앙 정렬 박스<br>b. **타이틀**:<br>- 상단 중앙에 **"계정 정보 설정"** 텍스트 노출<br>- H2 Scale, Bold 처리 |
| 3 | 입력 필드 그룹<br>(Form) | **3-1. 닉네임**<br>- 레이블: `닉네임`<br>- Input Placeholder: `사용할 닉네임`<br><br>**3-2. 이메일**<br>- 레이블: `이메일`<br>- Input Placeholder: `example@mail.com`<br>- Input Type: Email |
| 4 | 액션 버튼<br>(CTA Button) | a. **스타일**:<br>- 카카오 브랜드 컬러(Yellow)<br>- Full-width 버튼<br>- 라운드 처리<br>b. **텍스트**:<br>- 기본: `카카오 로그인으로 연결`<br>- 상황에 따라 `저장하기` 등으로 변경 가능<br>c. **상태**:<br>- 닉네임/이메일 유효성 검사 통과 전까지 **Disabled(비활성)** 처리 권장 |
