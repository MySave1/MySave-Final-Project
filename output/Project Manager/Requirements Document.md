## 2. 비기능적 요구사항 (Non-Functional Requirements)

### 성능 (Performance)

- AI 요약 생성은 요청 후 **5초 이내**에 완료되거나, 스트리밍 방식으로 응답성을 보장해야 한다.

### 보안 (Security)

- 사용자 비밀번호는 단방향 암호화(bcrypt 등) 방식으로 저장한다.
- JWT 토큰은 HttpOnly Cookie 방식으로 관리한다.

### 가용성 (Availability)

- 확장 프로그램은 오프라인 상태에서도 UI가 깨지지 않아야 하며, 저장 실패 시 사용자에게 안내해야 한다.
- 서비스 목표 가동률은 **99.9%**로 설정한다.

## 2. 비기능적 요구사항 (Non-Functional Requirements)

### 성능 (Performance)

- 북마크 저장 요청은 평균 **2초 이내**에 처리되어야 한다.
- 목록 조회 및 검색은 사용자 체감 지연이 없어야 한다.

### 보안 (Security)

- 사용자 인증은 OAuth 2.0 기반 소셜 로그인을 사용한다.
- JWT 토큰은 HttpOnly Cookie 방식으로 관리한다.
- 주요 API는 인증된 사용자만 접근 가능해야 한다.

### 가용성 (Availability)

- 확장 프로그램은 네트워크 오류 시에도 UI가 정상 동작해야 한다.
- 서버 장애 시 사용자에게 명확한 오류 메시지를 제공해야 한다.
- 목표 가동률은 **99.9%**로 설정한다.
