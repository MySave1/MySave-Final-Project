## 1. 프로젝트 개요 (Project Overview)

* **프로젝트 목표 (Goal):** 사용자 편의를 위한 웹 기반 북마크 관리 및 개인 대시보드 서비스(확장 프로그램 포함) 구축
* **범위 (Scope):** 기획/디자인부터 웹/확장 프로그램 개발, 서버 구축 및 배포, 최종 테스트까지의 전 과정
* **자원 (Resources):**
    * **인력:** 기획자, 디자이너, 프론트엔드 개발자, 백엔드 개발자 (4인 구성 가정)
    * **도구:** Notion (일정/이슈 관리), GitHub (형상 관리), Figma (디자인), AWS/GCP (서버)

---

## 2. 주요 산출물 목록 (Key Deliverables)

1.  **기획서 및 디자인:** 요구사항 정의서, UI/UX 디자인 시안, 확장 프로그램 아이콘
2.  **설계 문서:** ERD, 데이터베이스 스키마, API 명세서
3.  **소스 코드:** 프론트엔드(React/Vue 등), 백엔드(Node/Spring 등), 브라우저 확장 프로그램 코드
4.  **배포 결과물:** 접속 가능한 웹 사이트 URL, 설치 가능한 확장 프로그램 파일
5.  **최종 보고서:** 완료 보고서 및 발표 자료
